# Rental Analyzer Frontend

Beginner-friendly frontend MVP for analyzing rental deals.

## Commands

- Install: `npm install`
- Dev server: `npm run dev`
- Unit tests: `npm test`
- E2E tests: `npm run test:e2e`
- Build: `npm run build`

## MVP Flow

1. Property
2. Financing
3. Income
4. Expenses
5. Review and results

Main output metrics:

- Monthly cash flow
- Cash-on-cash return
- Deal score (0-100)
